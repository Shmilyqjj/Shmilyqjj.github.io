###HEXO目录结构
```yml
|-- _config.yml     全局配置（网站名称作者主题部署等）
|-- package.json    框架参数（框架依赖拆件及其版本）
|-- scaffolds       脚手架（通用MarkDown模板，新建文章时，hexo根据这个目录的文件进行构建）
|-- source          网页资源（css,js,images,文章等）
   |-- _posts       博客文章（写文章的地方）
|-- themes          主题目录
|-- .gitignore
|-- package.json
```
